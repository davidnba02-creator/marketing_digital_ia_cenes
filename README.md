- styles.css
- script.js
- assets/hero-uptc.jpg
- assets/campus-uptc.jpg
- assets/logo-cenes.png
- assets/logo-uptc.png

## Publicación

1. Crear un repositorio público nuevo.
2. Subir el contenido interno de esta carpeta.
3. Ir a Settings > Pages.
4. Seleccionar:
   - Deploy from a branch
   - main
   - /(root)

## Recomendación

Usar un nombre de repositorio limpio, por ejemplo:
`marketing-digital-ia-cenes`
